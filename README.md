# Portal Gampong Leubeu

Website ini adalah portal informasi desa untuk Gampong Leubeu, Kecamatan Pidie, Kabupaten Pidie, Provinsi Aceh.

## Isi Folder

- `blang-galang.html` – file HTML utama yang berisi seluruh konten situs.
- `index.html` – salinan `blang-galang.html` yang dapat digunakan langsung sebagai halaman utama di hosting.
- `images/` – folder yang berisi semua aset gambar untuk halaman kuliner.

## Cara Menjalankan Lokal

1. Pastikan file `index.html` dan folder `images/` berada di satu folder.
2. Buka `index.html` di browser, atau gunakan server lokal seperti XAMPP.

## Cara Hosting

1. Upload `index.html` dan folder `images/` ke direktori root situs web pada hosting.
2. Pastikan file tetap memiliki struktur folder yang sama.
3. Akses website dari URL hosting Anda.

## Catatan

- Semua konten sudah disimpan di file HTML tanpa dependensi eksternal selain gambar lokal.
- Jika ingin memperbarui konten, edit `blang-galang.html` lalu salin ulang ke `index.html`.
